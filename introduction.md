# Nobody Chat 是什么

Nobody Chat 是一个使用了 openaiGPT 的一款聊天工具, 具有以下功能:
* 记录聊天内容
* 支持语音输入以及朗读
* 支持笔记
* 支持自定义的 prompt

其他还在继续更新中...
